括号匹配问题：给定由左括号和右括号组成的字符串，求括号匹配的最长子串。

问题描述

给定由左括号和右括号组成的字符串，求括号匹配的最长子串。

输入格式

输入一行字符串，内容仅包括’(‘与’)’；

输入样例

)(()))()(()())

输出格式

输出该序列中括号匹配的最长字串

输出样例

()(()())

解题思路

用栈消除所有能匹配的括号，栈内保留不能匹配的括号，这些不能匹配的括号就是”断点“，计算得到两个断点间的最大距离就行（包括两个端点）

解题步骤

  1. 用数组作为底层写一个栈，实现几个基本的功能：入栈，出栈，计算栈长度，返回当前栈顶的值，返回当前栈指针。（这一步不是很有必要）
  2. 进行一次遍历，三种情况进行入栈1）栈空 2）当前字符为’(‘ 3）当前栈顶的值是’)’，其余情况出栈。栈内从小到大保留了所有”断点信息“
  3. 将字符串长度n入栈，代表右端点，设置如果空栈则返回-1，代表左端点。
  4. 遍历栈，比较栈内两两之差，找到两两之差最大的一对断点，以此为坐标输出字符串。

解题代码

import java.util.Scanner;
public class Step2{
    public static void main(String[] args) {
        Scanner scan = new Scanner(System.in);
        String str = scan.next();
        int n = str.length();
        Stack stack = new Stack(n);
        for (int i=0;i<n;i++){
            if (stack.isEmpty() || str.charAt(i)=='(' || str.charAt(stack.topValue())==')') {
                stack.push(i);
            }
            else {
                stack.pop();
            }
        }
        int rFlag = n;
        int lFlag = 0;
        int maxLen = 0;
        int numFor = stack.getTop();
        stack.push(n);
        for (int i = 0;i <= numFor+1 ; i++){
            int j = stack.pop();
            int k = stack.topValue();
            int tempLen = j-k;
            if (tempLen > maxLen && j-k>1){
                maxLen = tempLen;
                rFlag = j;
                lFlag = k;
            }
        }
        System.out.print(str.substring(lFlag+1,rFlag));
    }
}

class Stack {
    private int stackSize;
    private int[] stack;
    private int top = -1;
    public Stack2(int stackSize) {
        this.stackSize = stackSize;
        stack = new int[this.stackSize];
    }
    public boolean isFull() {
        return top == stackSize - 1;
    }
    public boolean isEmpty() {
        return top == -1;
    }
    public void push(int value) {
        if (isFull()) {
            System.out.println("栈满");
            return;
        }
        top++;
        stack[top] = value;
    }
    public int pop() {
        if (isEmpty()) {
            throw new RuntimeException("栈空，没有数据~");
        }
        int value = stack[top];
        top--;
        return value;
    }
    public int topValue() {
        if (isEmpty()){
            return -1;
        }
        return stack[top];
    }
    public int getTop(){
        return top;
    }
    public void clear(){top=-1;}
}
