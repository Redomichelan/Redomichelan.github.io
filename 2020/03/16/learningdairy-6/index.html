解题部分

Reverse sign in

题目

查看main函数伪代码如下

__int64 __fastcall main(__int64 a1, char **a2, char **a3)
{
  __int64 result; // rax
  char s; // [rsp+0h] [rbp-30h]
  unsigned __int64 v5; // [rsp+28h] [rbp-8h]

  v5 = __readfsqword(0x28u);
  printf("Please input your flag:", a2, a3);
  __isoc99_scanf("%32s", &s);
  if ( strlen(&s) == 32 )
  {
    if ( (unsigned int)sub_400686((__int64)&s) )
      puts("Right!");
    else
      puts("Wrong!");
    result = 0LL;
  }
  else
  {
    puts("Wrong!");
    result = 0LL;
  }
  return result;
}

解题思路

根据伪代码我们可以进行初步分析，在这道题中需要键入一串32个字符的字符串，然后执行sub_400686函数，如果执行结果为真，则输出right。
查看sub_400686函数伪代码。

signed __int64 __fastcall sub_400686(__int64 a1)
{
  signed int i; // [rsp+Ch] [rbp-Ch]

  for ( i = 0; i <= 31; ++i )
  {
    if ( (char)(*(_BYTE *)(i + a1) ^ byte_400818[i]) != i )
      return 0LL;
  }
  return 1LL;
}

可以看到，sub_400686的过程是，在每一次循环中将参数a1的每一位字符与byte_400818的字符进行异或运算，如果其结果不等于i，则返回0。
整理信息可以发现，如果要是结果最后输出right，则需要每次异或运算均成立。查看数据得到16进制数组。

byte_400818=[66,6D,63,64,7F,3C,36,72,57,42,64,3B,7B,52,7C,3C,66,54,60,60,27,4A,49,7F,71,58,52,72,7D,75,2A,62]

得到以上信息，就可以构建脚本了。

解题过程

byte=[0x66,0x6D, 0x63, 0x64, 0x7F, 0x3C, 0x36, 0x72, 0x57, 0x42, 0x64,0x3B, 0x7B, 0x52,
      0x7C, 0x3C, 0x66, 0x54, 0x60,0x60, 0x27,0x4A, 0x49, 0x7F, 0x71,
      0x58, 0x52, 0x72, 0x7D, 0x75, 0x2A,0x62]

for i in range(32):
    print(chr(byte[i]^i))

得到flag。

junior math

题目

查看main函数伪代码如下

__int64 __fastcall main(__int64 a1, char **a2, char **a3)
{
  printf("Please input your flag:", a2, a3);
  __isoc99_scanf("%32s", &byte_602080);
  sub_4005D6();
  sub_40063F();
  sub_4006A8();
  sub_400711();
  sub_40077A();
  sub_4007E3();
  sub_40084C();
  sub_4008B5();
  sub_400918();
  sub_400981();
  sub_4009EA();
  sub_400A53();
  sub_400ABC();
  sub_400B25();
  if ( dword_602064 )
    puts("Wrong!");
  else
    puts("Right!");
  return 0LL;
}

解题思路

初步逻辑，键入字符串，进行sub_4005D6()等一系列函数运算，最后参数 dword_602064 值为0即可输出right。
查看函数sub_4005D6()函数,伪代码如下

__int64 sub_4005D6()
{
  _BOOL4 v0; // edx
  __int64 result; // rax

  v0 = (byte_602080 - 147) * byte_602080 != -4590 || (byte_602082 - 147) * byte_602082 != -4850;
  result = v0 | (unsigned int)dword_602064;
  dword_602064 |= v0;
  return result;
}

其后的函数都是同一个类型，变化的只有式子中的常数和运算的字符。
整体分析，如果需要最后的 dword_602064 为0 则需要保证每一步函数计算的最后一步与运算为0，这需要两个参数均为0，即v0 = 0。
要使得 v0 = 0，则令输入的字符串满足一元二次方程即可。
首先，因为flag{}的格式固定，所以我们可以从第6个字符开始计算

解题过程

观察函数，提取每个字符运算当中对应的b，c值存入数组。
计算一元二次方程，代码如下

import math

b=[-214,-1111,-1010,-741,-114,-911,-2333,-6666,-917,-1020,-54321]
c=[10153,51024,99789,45764,1805,87418,118612,759800,84552,68256,6774500]

for i in range(11):
  derta = b[i] * b[i] - 4  * c[i]
  X = (-b[i] - math.sqrt(derta)) / 2
  print(chr(int(X)))

最后运算的结果有一位字符出现了乱码，但是小问题很好解决，有需要复制代码的同学自行思考。
最后拼接我们省略的部分，得到flag。

学习心得

因为没有系统学习过汇编语言，逆向的部分一直是比较困难的地方（这个困难还会延续到pwn），但好在我有c语言的基础，遇到的很多基础问题也比较容易解决，比如存入数据之后的地址偏移什么的。最后非常开心能完成这两道简单的逆向题，前方虽然困难重重，但是我们要相信，克服困难的最好办法就是微笑着面对它，加油，奥里给！
