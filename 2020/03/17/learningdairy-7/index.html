解题过程

Another 01Game

解题过程

题目没有前两道题直白，但是提示比较全面。照着提示将代码加工成二维码，扫描不出来，比较后发现是黑白反转，可以将代码中0，1互换，再进行生成，扫描得到一串二进制代码，根据提示发现是7位二进制的asc码，解密即得到flag。
贴两段代码

from PIL import Image

MAX = 37
pic = Image.new("RGB",(MAX, MAX))
str='11------------1001'
i=0
for y in range (0,MAX):
    for x in range (0,MAX):
        if(str[i] == '1'):
            pic.putpixel([x,y],(0, 0, 0))
        else:
            pic.putpixel([x,y],(255,255,255))
        i = i+1
pic.show()
pic.save("flag.png")

PkCrack

解题过程

提示中的解密工具貌似只有32位，于是下载了Advanced Archive Password Recovery。选择明文攻击和plan，解压。提交

Birthday

解题过程

使用之前下载的Advanced Archive Password Recovery暴力破解，设定8位数密码，纯数字，很快解压。提交

NTFS

解题过程

Invisible flag

解题过程

图片被遮住一节，提示注意高度，用hex打开编辑高度，得到flag。提交。

BabyBase

解题过程

查看一串纯数字，没规律，提示是base加密。于是逆向思考。将“flag”字符串多次加密发现base16多次加密可以出现全是数字的效果，于是写了个脚本解base16，但是失败了。因为不只是base16加密，出现报错。考虑到本人的代码能力和加密次数不多，于是手动复制解密。得到flag

学习心得

今天更新的部分是misc里除开流量分析的内容，因为提示的很多（看说明是给新手练手），所以更新的这部分不是很困难，当然有时候也会有思维盲区的时候，比如BabyBase的题目我在开始就不清楚是怎么回事，直到后面自己尝试多次加密之后才发现问题，要不然我可能一直发现不了多次base16加密有那个效果。感觉这部分的内容总体还是比较轻松，但是收获也是很多的。
