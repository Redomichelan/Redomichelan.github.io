这是一个选择结构。我们开始需要输入两个数字，分别储存在v9和v10两个变量中。

我们先随便取一个值带进去算一下。

phase_4

解题过程

查看阶段四伪代码

int __cdecl phase_4(int a1)
{
  int result; // eax
  int v2; // [esp+18h] [ebp-14h]
  int v3; // [esp+1Ch] [ebp-10h]

  if ( __isoc99_sscanf(a1, "%d %d", &v3, &v2) != 2 || (unsigned int)(v2 - 2) > 2 )
    explode_bomb();
  result = func4(5, v2);
  if ( result != v3 )
    explode_bomb();
  return result;
}

这其中有一个func4函数，伪代码如下。

int __cdecl func4(int a1, int a2)
{
  int result; // eax
  int v3; // edi

  if ( a1 <= 0 )
    return 0;
  result = a2;
  if ( a1 != 1 )
  {
    v3 = func4(a1 - 1, a2) + a2;
    result = v3 + func4(a1 - 2, a2);
  }
  return result;
}

发现这是一个嵌套函数。
首先分析基本的信息得到，必须输入两个值分别存入，且v2小于等于4。
在此基础上将v2进行函数运算，返回值等于v3的值。
嵌套函数分析有点绕，所以我选择在python中将函数构建出来。

        result = v3+func4(a1-2,a2);
    return result;
