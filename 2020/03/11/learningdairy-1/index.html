本系列是博主在OUC进修前做新生测试培训的一些心得、思路和学习体会。希望能在新的人生阶段更加努力地进步，获得自己的成长与成就。

解题部分

welcome to web

题目

<?php
	if(isset($_GET['key'])){
		if($_GET['key'] == 'areyousure'){
			echo 'For this exercise, flag is: ******';
		}
	}
?>

解题思路

本题包含两个if判断，只有当两个if判断均为true的时候，才能得到“flag is：” 后面隐藏的信息。
显而易见，要另两个if判断均为true需要满足‘key’的值不为NULL且等于’areyousure’。
web构建请求可以通过GET和POST两种方式，在本题中可以通过构建url提交GET请求来设定‘key’的参数值以得到flag。
在顶部url后键入

?key=areyousure

得到flag。

Calculator

题目

url：http://vps1.blue-whale.me:23331/calculator/
[]内指代随机数

Let's play a game. Please work on this math problem, and make it in 1.5 seconds.

[] + [] * [] - [] =

解题思路

观察知，本题包含4个随机数，需要在4个随机数在服务器刷新后的1.5秒内提交正确答案。
常规思路即写个脚本爬取算式，计算后再提交答案。
在空格随意输入并提交观察url变化，易得本题可通过GET请求设定参数。
初步构思，通过爬虫访问地址，爬取随机数并计算，然后构建url传递参数，得到flag。

代码
import requests
from bs4 import BeautifulSoup
/*引入requests和bs4库*/

url ="http://vps1.blue-whale.me:23331/calculator/"
s = requests.session()/*创建session保持两次访问cookie一致*/
r = s.get(url)

soup = BeautifulSoup(r.text,'html.parser')
kl = soup.span.string/*提取特定标签中的算式*/
kl = kl.strip().strip('=')/*去除算式最后的等号*/

ans =  eval(kl)/*直接计算字符串并返回值*/

url1 ='http://vps1.blue-whale.me:23331/calculator/?answer=%s' % (ans)

r2 = s.get(url1)

print(r2.text)/*打印页面查看flag*/

学习心得

这是我第一次发技术博客，也是好久没有正儿八经地学习后第一次做ctf的题目。在做第二个题目的时候其实也遇到过挺多问题，比如提取标签内的算式是字符串，不知道怎么处理，在朋友的指导下我第一次使用了eval语法，感觉嘛只有一个，python天下第一！
虽然这次新生培训测试只在热身阶段，但是我觉得还是有点难的，可能是我的基础比较薄弱。我也会好好珍惜这次机会，努力学习。
当你在凝视大佬的时候，有一天或许也会被这么凝视。
愿望总是要有的。
